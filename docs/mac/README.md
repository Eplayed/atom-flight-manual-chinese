# 起航

Atom 是 Github 出品的文本编辑器，同时它也是一款可以 hackable 的 21 世纪现代化文本编辑器。由于官方没有相关的中文文档，Atom 中国社区对于 Atom 相关的文档资料甚少，有些翻译也不是很准确。因此我发起了这个中文翻译文档。
